Falta configurar Supabase

Abre el archivo config.js y pega la URL y la clave pública (anon/publishable) de tu proyecto de Supabase.

window.SUPABASE_CONFIG = {
  url: "https://xxxx.supabase.co",
  key: "eyJhbGci... (clave pública)"
};

Las encuentras en tu panel de Supabase, en Project Settings → API. Usa la clave pública (anon / publishable), nunca la secreta.
